**Performance:**
- Image generation: ~8-10s each (DreamShaper XL)
- SVD animation: ~45-60s per clip (the bottleneck)

**VRAM Management — Critical:**
1. Generate ALL 3 images BEFORE running SVD
2. Run SVD sequentially, one at a time
3. Send ComfyUI interrupt after each phase to free 1-2GB overhead
4. FFmpeg runs on CPU only
5. Wait 5s after SVD before FFmpeg for VRAM settle
6. Abort if VRAM < 2GB free
